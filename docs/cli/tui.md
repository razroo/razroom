# `razroom tui`

Open the terminal UI connected to the Gateway.

Related:

- TUI guide: [TUI](/web/tui)

## Examples

```bash
razroom tui
razroom tui --url ws://127.0.0.1:18789 --token <token>
razroom tui --session main --deliver
```
